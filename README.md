# date-helper

A lightweight JavaScript/TypeScript library for **date manipulation, formatting and validation**.

Inspired by the excellent library **date-fns**, this project was created to explore how date abstractions work internally and provide a simple and efficient API for common date operations.

## Features

- 📅 Date formatting
- ➕ Add days, months and years
- 🔎 Date validation
- ⚖️ Date comparison
- 📆 Day of week formatting
- 🔄 Date parsing from multiple input types
